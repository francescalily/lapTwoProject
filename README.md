# lapTwoProject
Lap Two Project - A community app aimed for Florin council

## To run server
- make sure you are in the root directory (not in /api or /client)
- make a .env file
- add
    ```
    PORT=3000
    DB_URL='<your elephantSQL URL>'
    BCRYPT_SALT_ROUNDS = 12
    TOKEN_KEY = 'randomString' 
    ```
- run ```npm run start```
